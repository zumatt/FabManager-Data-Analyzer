# FabManager Data Analyzer

The project has started as a collection of Python notebooks for reading data from FabManager using APIs.

[![View FabManager-Scraping on GitHub](https://img.shields.io/github/stars/zumatt/FabManager-Scraping?color=232323&label=FabManager-Scraping&logo=github&labelColor=232323)](https://github.com/zumatt/FabManager-Scraping)

The actual version is being developed as a package to perform the following actions:
- Extract data from FabManager:
   - Export accounting data (coming soon...);
   - Export bookable machines data (coming soon...);
   - Export events dat (coming soon...)a;
   - Export invoices data (coming soon...);
   - Export machines data;
   - Export plan categories data (coming soon...);
   - Export plans data (coming soon...);
   - Export prices data (coming soon...);
   - Export reservations data;
   - Export spaces data (coming soon...);
   - Export subscriptions data (coming soon...);
   - Export trainings data;
   - Export user trainings data (coming soon...);
   - Export users data;
- Export data from FabManager for being published as [Open Data](https://en.wikipedia.org/wiki/Open_data):
   - Export reservation data anonymised to be published as [Linked Open Data](https://en.wikipedia.org/wiki/Linked_data) in [JSON-LD](https://json-ld.org/) format;
   - Export reservation data anonymised to be published as Open Data in .json format;
   - Export reservation data pseudo-anonymised to be published as [Linked Open Data](https://en.wikipedia.org/wiki/Linked_data) in [JSON-LD](https://json-ld.org/) format;
   - Export reservation data pseudo-anonymised to be published as Open Data in .json format;
- Analyse data from FabManager:
   - Analyse reservation in comparison with trainings data (coming soon...).
